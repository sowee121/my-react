### `npm start`

开发模式下运行程序.<br>
运行地址： [http://localhost:3000](http://localhost:3000).

监控文件修改，将自动热加载.<br>

### `npm test`

在交互监视模式下启动测试运行程序.


### `npm run build`

编译打包程序，发布生产环境，并放在build目录中.
